I
Notion de vecteurTranslation et caractéristiques

En géométrie, une translation déplace tous les points du plan d'une même façon : même direction, même sens, même distance. Le vecteur est l'objet mathématique qui décrit ce déplacement.

Définition — Vecteur

Un vecteur est caractérisé par trois éléments :

📐 Direction

L'orientation de la droite portant le vecteur
(horizontale, verticale, oblique…)

➡️ Sens

Le sens de déplacement sur cette droite
(vers la droite, vers le haut, etc.)

📏 Norme

La longueur du vecteur, notée ‖u⃗‖
(toujours positive ou nulle)

🔤 Notation

On note un vecteur avec une lettre et une flèche : u⃗, v⃗, AB⃗
AB⃗ va du point A vers le point B

Définition — Vecteur AB⃗

Si A et B sont deux points du plan, le vecteur AB⃗ est le vecteur qui représente la translation qui envoie A sur B. Son point de départ est A (appelé origine) et son point d'arrivée est B (appelé extrémité).

Représentation d'un vecteur

Définition — Vecteur nul

Le vecteur nul, noté 0⃗, est le vecteur de norme zéro. Il correspond à une translation de longueur nulle : AA⃗ = 0⃗ pour tout point A.

Définition — Vecteur opposé

Le vecteur opposé de u⃗ est le vecteur noté −u⃗, de même direction et même norme, mais de sens contraire. On a : AB⃗ = −BA⃗

II
Égalité de vecteursQuand deux vecteurs sont-ils égaux ?
Définition — Égalité de vecteurs

Deux vecteurs sont égaux s'ils ont la même direction, le même sens et la même norme. On écrit u⃗ = v⃗.
Remarque : un vecteur n'est pas attaché à un point précis, il peut être représenté n'importe où dans le plan.

Propriété — Parallélogramme

AB⃗ = DC⃗ si et seulement si ABCD est un parallélogramme (ou un rectangle, un losange, un carré, qui sont des cas particuliers de parallélogramme).

Exemple — Parallélogramme ABCD

Dans le parallélogramme ABCD :

• AB⃗ = DC⃗   • AD⃗ = BC⃗   • AC⃗ ≠ BD⃗ (sauf si c'est un rectangle particulier)

Les diagonales d'un parallélogramme se coupent en leur milieu.

Méthode — Vérifier si AB⃗ = CD⃗
  1. Calculer les coordonnées de AB⃗ : (x_B − x_A ; y_B − y_A)
  2. Calculer les coordonnées de CD⃗ : (x_D − x_C ; y_D − y_C)
  3. Si les deux couples de coordonnées sont identiques → AB⃗ = CD⃗
III
Addition de vecteursRègle de Chasles et règle du parallélogramme
Règle de Chasles

Pour tous points A, B, C du plan :

AB⃗ + BC⃗ = AC⃗

La règle de Chasles permet de « chaîner » des vecteurs : le point intermédiaire B se simplifie.

Exemples d'application de Chasles

• AB⃗ + BC⃗ + CD⃗ = AD⃗

• AM⃗ + MB⃗ = AB⃗

• AC⃗ + CB⃗ = AB⃗

• AB⃗ + BA⃗ = AA⃗ = 0⃗ (un vecteur et son opposé s'annulent)

Règle du parallélogramme

La somme u⃗ + v⃗ est la diagonale du parallélogramme construit sur u⃗ et v⃗ (placés avec la même origine).

Construction de u⃗ + v⃗ — Règle du parallélogramme

Propriétés de l'addition de vecteurs
  • Commutativité : u⃗ + v⃗ = v⃗ + u⃗
  • Associativité : (u⃗ + v⃗) + w⃗ = u⃗ + (v⃗ + w⃗)
  • Élément neutre : u⃗ + 0⃗ = 0⃗ + u⃗ = u⃗
  • Opposé : u⃗ + (−u⃗) = 0⃗
  • Soustraction : AB⃗ − AC⃗ = CB⃗ (on inverse le signe → on inverse le sens)
IV
Multiplication par un scalaireÉtirer, réduire, retourner un vecteur
Définition — Produit k · u⃗

Soit k un réel et u⃗ un vecteur non nul. Le vecteur k · u⃗ est :

Condition sur kDirectionSensNorme
k > 0Même que u⃗Même sens|k| × ‖u⃗‖
k < 0Même que u⃗Sens opposé|k| × ‖u⃗‖
k = 0——0 (vecteur nul)
Exemples

• 2u⃗ : même direction, même sens, norme doublée

• −u⃗ : même direction, sens opposé, même norme (= vecteur opposé)

• ½ u⃗ : même direction, même sens, norme divisée par 2

• −3u⃗ : même direction, sens opposé, norme triplée

Propriétés — Distributivité
  • k(u⃗ + v⃗) = ku⃗ + kv⃗
  • (k + l)u⃗ = ku⃗ + lu⃗
  • (kl)u⃗ = k(lu⃗)
  • 1 · u⃗ = u⃗
Définition — Vecteurs colinéaires

Deux vecteurs u⃗ et v⃗ sont colinéaires si et seulement si il existe un réel k tel que v⃗ = k · u⃗ (l'un est multiple de l'autre). Le vecteur nul est colinéaire à tout vecteur.

V
Coordonnées d'un vecteurDans un repère du plan

Dans un repère (O ; i⃗, j⃗) du plan, tout vecteur peut être décrit par deux nombres : ses coordonnées.

Définition — Coordonnées d'un vecteur

Si A(x_A ; y_A) et B(x_B ; y_B) sont deux points, alors :

AB⃗ = ( x_B − x_A ; y_B − y_A )
⚠️

Sens du calcul : on fait toujours arrivée − départ. Pour AB⃗, on fait B − A (et non A − B).

Exemple

A(2 ; 5) et B(7 ; 1) :

AB⃗ = (7 − 2 ; 1 − 5) = (5 ; −4)

BA⃗ = (2 − 7 ; 5 − 1) = (−5 ; 4) = −AB⃗ ✓

Opérations sur les coordonnées
  • Si u⃗ = (a ; b) et v⃗ = (c ; d) alors u⃗ + v⃗ = (a+c ; b+d)
  • Si u⃗ = (a ; b) et v⃗ = (c ; d) alors u⃗ − v⃗ = (a−c ; b−d)
  • Si u⃗ = (a ; b) et k ∈ ℝ alors ku⃗ = (ka ; kb)
  • Norme : ‖u⃗‖ = √(a² + b²) (théorème de Pythagore)
‖AB⃗‖ = √( (x_B − x_A)² + (y_B − y_A)² )
Calcul de norme

u⃗ = (3 ; 4) → ‖u⃗‖ = √(3² + 4²) = √(9 + 16) = √25 = 5

v⃗ = (1 ; 1) → ‖v⃗‖ = √(1 + 1) = √2 ≈ 1,41

Méthode — Milieu d'un segment
  1. Si M est le milieu de [AB], alors AM⃗ = ½ AB⃗
  2. Les coordonnées de M sont : x_M = (x_A + x_B)/2 et y_M = (y_A + y_B)/2
Exemple — Milieu

A(2 ; 6) et B(8 ; 2) :

M = ((2+8)/2 ; (6+2)/2) = (5 ; 4)

Vérification : AM⃗ = (3 ; −2) et MB⃗ = (3 ; −2) → AM⃗ = MB⃗ ✓

VI
ColinéaritéAlignement de points et droites parallèles
Définition — Déterminant de deux vecteurs

Si u⃗ = (a ; b) et v⃗ = (c ; d), le déterminant (ou produit vectoriel en 2D) de u⃗ et v⃗ est :

det(u⃗, v⃗) = a × d − b × c
Propriété fondamentale — Critère de colinéarité

u⃗ = (a ; b) et v⃗ = (c ; d) sont colinéaires si et seulement si :

a × d − b × c = 0

Autrement dit, det(u⃗, v⃗) = 0 ⟺ u⃗ et v⃗ colinéaires

Propriété — Points alignés

Trois points A, B, C sont alignés si et seulement si AB⃗ et AC⃗ sont colinéaires, c'est-à-dire si :

det(AB⃗, AC⃗) = 0

Méthode — Vérifier l'alignement de trois points
  1. Calculer les coordonnées de AB⃗ = (x_B − x_A ; y_B − y_A)
  2. Calculer les coordonnées de AC⃗ = (x_C − x_A ; y_C − y_A)
  3. Calculer le déterminant : det = (x_B−x_A)(y_C−y_A) − (y_B−y_A)(x_C−x_A)
  4. Si det = 0 → points alignés · Si det ≠ 0 → points non alignés
Exemple — Alignement

A(1 ; 2), B(3 ; 6), C(5 ; 10). Les points sont-ils alignés ?

AB⃗ = (2 ; 4) · AC⃗ = (4 ; 8)

det = 2 × 8 − 4 × 4 = 16 − 16 = 0

→ det = 0 donc A, B, C sont alignés ✓

Exemple — Non alignés

A(0 ; 0), B(2 ; 1), C(3 ; 4). Les points sont-ils alignés ?

AB⃗ = (2 ; 1) · AC⃗ = (3 ; 4)

det = 2 × 4 − 1 × 3 = 8 − 3 = 5 ≠ 0

→ det ≠ 0 donc A, B, C ne sont pas alignés

Propriété — Droites parallèles

Deux droites (AB) et (CD) sont parallèles (ou confondues) si et seulement si AB⃗ et CD⃗ sont colinéaires :
det(AB⃗, CD⃗) = 0 ⟺ (AB) ∥ (CD)

⭐ À retenir — Bilan du chapitre

  • Un vecteur est défini par sa direction, son sens et sa norme
  • Chasles : AB⃗ + BC⃗ = AC⃗ (simplification du point intermédiaire)
  • Coordonnées : AB⃗ = (x_B − x_A ; y_B − y_A) → toujours arrivée − départ
  • Norme : ‖u⃗‖ = √(a² + b²)
  • Colinéarité : det(u⃗, v⃗) = ad − bc = 0 ↔ u⃗ et v⃗ colinéaires ↔ points alignés ou droites parallèles
  • Milieu M de [AB] : x_M = (x_A + x_B)/2, y_M = (y_A + y_B)/2
Exercice d'application n°1 — Coordonnées

On donne A(1 ; 3), B(4 ; 7), C(−1 ; 5). Calculer AB⃗, AC⃗ et ‖AB⃗‖.

Voir la correction

• AB⃗ = (4−1 ; 7−3) = (3 ; 4)

• AC⃗ = (−1−1 ; 5−3) = (−2 ; 2)

• ‖AB⃗‖ = √(3² + 4²) = √(9+16) = √25 = 5

Exercice d'application n°2 — Chasles

Simplifier l'expression : AB⃗ + BC⃗ + CD⃗ + DE⃗

Voir la correction

• AB⃗ + BC⃗ = AC⃗ (Chasles)

• AC⃗ + CD⃗ = AD⃗ (Chasles)

• AD⃗ + DE⃗ = AE⃗ (Chasles)

Résultat final : AB⃗ + BC⃗ + CD⃗ + DE⃗ = AE⃗

Exercice d'application n°3 — Colinéarité

A(2 ; 1), B(5 ; 3), C(8 ; 5). Ces trois points sont-ils alignés ?

Voir la correction

• AB⃗ = (3 ; 2) · AC⃗ = (6 ; 4)

• det(AB⃗, AC⃗) = 3 × 4 − 2 × 6 = 12 − 12 = 0

→ det = 0 → A, B et C sont alignés.

On remarque aussi que AC⃗ = 2 × AB⃗ (les vecteurs sont colinéaires de rapport 2).
